Frequently asked questions.

About the Bot.

This bot was developed to improve server management, offering features including channel creation and message management.

What are the main commands?

Use .embed to create embedded messages and .delete to clear messages.

How do I moderate my members?

Use .ban and .expel to ban or expel members, respecting the necessary permissions.

How do I obtain member information?

The .info command provides details about members, including name, nickname, ID, and joining and creation dates.

How do I create a channel?

Use .createrc to create text channels and .createv for voice channels, ensuring appropriate permissions.

How to deal with roles?

Assign or remove roles with .addcargo and .remcargo, requiring appropriate permissions.

How do I see the bot's ping?

Check latency with .ping, displaying the response in milliseconds.

Why use Orbit?

Orbit simplifies management, offering efficient features such as channel creation, quick moderation and personalized interactions. Try it to improve your server efficiency and customization!
